# Ejercicio 3: Termostato Inteligente

## 📝 Descripción
Crea una clase `Termostato` que mantenga la temperatura de una habitación. Debe tener un objetivo de temperatura (setpoint) y modos: "Enfriar", "Calentar" y "Apagado". El termostato debe decidir su acción basándose en la temperatura actual.

> **Contexto Académico**: Introduce la lógica de control reactivo, donde el comportamiento del objeto depende de la comparación entre un estado deseado y un estado actual.

## 🎯 Objetivos de Aprendizaje
- Uso de constantes o valores fijos para modos de operación.
- Lógica de comparación continua.
- Simulación de sensores y actuadores.

## 📥 Ejemplo de Entrada
```text
Temp Actual: 25°C, Objetivo: 20°C, Modo: Automático
Acción: Evaluar sistema
```

## 📤 Ejemplo de Salida
```text
Sistema en modo ENFRIAR (Temp actual > Objetivo).
```

---
🕓 **Dificultad**: Difícil
📚 **Temas**: Sistemas de control, Lógica comparativa.
