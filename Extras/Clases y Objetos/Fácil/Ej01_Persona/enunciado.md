# Ejercicio 1: Ficha Personal

## 📝 Descripción
Se requiere un sistema básico para registrar la información de ciudadanos. Debes diseñar una clase que represente a una persona, permitiendo almacenar sus datos fundamentales y mostrar una presentación formal de la misma.

> **Contexto Académico**: Este ejercicio introduce la estructura fundamental de una clase: atributos privados, constructor para inicialización y métodos de comportamiento.

## 🎯 Objetivos de Aprendizaje
- Definición de una clase y creación de objetos (instanciación).
- Uso de modificadores de acceso (`private` y `public`).
- Implementación de constructores para inicializar el estado del objeto.
- Uso de la palabra clave `this`.

## 📥 Ejemplo de Entrada
```text
// Datos pasados al constructor en el main:
Nombre: "Ana García"
Edad: 25
Ciudad: "Madrid"
```

## 📤 Ejemplo de Salida
```text
=== FICHA PERSONAL ===
Nombre: Ana García
Edad: 25 años
Ciudad: Madrid
```

---
🕓 **Dificultad**: Fácil
📚 **Temas**: Clases, Atributos, Constructor, Métodos.
