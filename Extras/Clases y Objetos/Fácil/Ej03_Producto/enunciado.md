# Ejercicio 3: Control de Inventario Simple

## 📝 Descripción
Diseña una clase para gestionar productos individuales en una tienda. Cada producto tiene un nombre, un precio y una cantidad en stock. El sistema debe permitir realizar ventas (reducir stock) y reabastecimientos (aumentar stock).

> **Contexto Académico**: Introduce la manipulación controlada del estado interno mediante métodos que reciben parámetros y aplican cambios.

## 🎯 Objetivos de Aprendizaje
- Gestión del estado de un objeto a través del tiempo.
- Uso de parámetros en métodos.
- Lógica condicional dentro de métodos de clase.

## 📥 Ejemplo de Entrada
```text
Producto: "Teclado", Precio: 25.0, Stock: 10
Venta: 3 unidades
Compra: 5 unidades
```

## 📤 Ejemplo de Salida
```text
Producto: Teclado | Stock Actual: 12 unidades
```

---
🕓 **Dificultad**: Fácil
📚 **Temas**: Métodos con parámetros, Modificación de atributos.
