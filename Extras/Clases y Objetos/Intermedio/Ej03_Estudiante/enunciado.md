# Ejercicio 3: Registro Académico

## 📝 Descripción
Diseña una clase `Estudiante` que almacene el nombre, la matrícula y tres calificaciones. El sistema debe ser capaz de calcular el promedio y determinar si el estudiante aprobó según el estándar de la institución (nota >= 4.0).

> **Contexto Académico**: Este ejercicio combina el manejo de múltiples atributos numéricos y la generación de estados derivados (aprobado/reprobado).

## 🎯 Objetivos de Aprendizaje
- Manejo de múltiples atributos del mismo tipo.
- Implementación de métodos que generan información nueva a partir de los datos existentes.
- Formateo básico de resultados.

## 📥 Ejemplo de Entrada
```text
Estudiante: "Sofía", Notas: 5.5, 3.8, 6.0
```

## 📤 Ejemplo de Salida
```text
Estudiante: Sofía (ID: 2024-001)
Promedio: 5.1
Estado: APROBADO
```

---
🕓 **Dificultad**: Intermedio
📚 **Temas**: Cálculos, Lógica de decisión, Métodos descriptivos.
