# Ejercicio 2: Matriz Dinámica (Tablero)

## 📝 Descripción
Representa un tablero de juego de tamaño variable utilizando una lista de listas (`ArrayList<ArrayList<Integer>>`). El programa debe inicializar el tablero con ceros, permitir colocar un valor en una coordenada (X,Y) y mostrar el tablero formateado como una cuadrícula.

> **Contexto Académico**: Introduce la anidación de colecciones, lo cual es la base para estructuras de datos complejas. Requiere doble iteración y manejo de índices en dos niveles.

## 🎯 Objetivos de Aprendizaje
- Declaración e inicialización de `ArrayList<ArrayList<T>>`.
- Acceso bidimensional dinámico.
- Doble ciclo para visualización matricial.

## 📥 Ejemplo de Entrada
```text
Tamaño: 3x3
Acción: Colocar 1 en (1, 1)
```

## 📤 Ejemplo de Salida
```text
0 0 0
0 1 0
0 0 0
```

---
🕓 **Dificultad**: Difícil
📚 **Temas**: Listas anidadas, Matrices dinámicas.
