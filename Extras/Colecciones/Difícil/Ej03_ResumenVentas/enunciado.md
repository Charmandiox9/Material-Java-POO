# Ejercicio 3: Resumen de Ventas por Categoría

## 📝 Descripción
Tienes una lista de objetos `Venta` (producto, categoría, monto). Debes procesar esta lista para generar un resumen que indique el total de ventas por categoría.

> **Contexto Académico**: Este ejercicio requiere coordinar información de una lista para realizar agrupaciones lógicas. Se debe pensar en cómo identificar categorías nuevas y acumular sus totales en listas paralelas o estructuras similares.

## 🎯 Objetivos de Aprendizaje
- Agrupación lógica de datos.
- Sincronización entre listas (Lista de Categorías y Lista de Totales).
- Lógica de búsqueda y actualización de acumuladores.

## 📥 Ejemplo de Entrada
```text
Ventas: 
("Pan", "Alimento", 10.0), 
("Jabón", "Aseo", 5.0), 
("Leche", "Alimento", 12.0)
```

## 📤 Ejemplo de Salida
```text
Resumen:
Alimento: $22.0
Aseo: $5.0
```

---
🕓 **Dificultad**: Difícil
📚 **Temas**: Agrupación, Acumulación condicional.
