# Ejercicio 1: Lista de Tareas Pendientes

## 📝 Descripción
Desarrolla un programa que permita a un usuario gestionar una lista de tareas. El usuario debe poder agregar nuevas tareas (Strings) y ver todas las tareas ingresadas con su respectivo número de orden.

> **Contexto Académico**: Este ejercicio introduce la estructura `ArrayList` y sus métodos fundamentales: `add()` para insertar y el uso de un ciclo `for` para recorrer la colección.

## 🎯 Objetivos de Aprendizaje
- Declaración e instanciación de un `ArrayList<String>`.
- Uso del método `add()` para dinamicidad.
- Recorrido de colecciones mediante su tamaño (`size()`).

## 📥 Ejemplo de Entrada
```text
Tareas: "Estudiar POO", "Comprar café", "Lavar ropa"
```

## 📤 Ejemplo de Salida
```text
=== MIS TAREAS ===
1. Estudiar POO
2. Comprar café
3. Lavar ropa
```

---
🕓 **Dificultad**: Fácil
📚 **Temas**: ArrayList, add(), size().
