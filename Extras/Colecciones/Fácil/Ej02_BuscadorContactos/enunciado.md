# Ejercicio 2: Buscador de Contactos

## 📝 Descripción
Crea una lista con nombres de amigos. El programa debe solicitar un nombre al usuario y determinar si ese nombre se encuentra en la lista, indicando en qué posición (índice) está.

> **Contexto Académico**: Se enfoca en la búsqueda de elementos y el uso de métodos de comparación de contenido (`contains` o búsqueda manual).

## 🎯 Objetivos de Aprendizaje
- Uso del método `contains()` o búsqueda secuencial.
- Manejo de índices con `indexOf()`.
- Lógica de búsqueda en colecciones.

## 📥 Ejemplo de Entrada
```text
Lista: ["Ana", "Pedro", "Luis"]
Buscar: "Pedro"
```

## 📤 Ejemplo de Salida
```text
El contacto 'Pedro' fue encontrado en la posición 1.
```

---
🕓 **Dificultad**: Fácil
📚 **Temas**: Búsqueda, indexOf(), contains().
