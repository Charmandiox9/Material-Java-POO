# Ejercicio 3: Calculadora de Gastos

## 📝 Descripción
Crea una lista de tipo `Double` que almacene los precios de productos en un carrito de compras. El programa debe recorrer la lista para calcular el costo total y mostrar cuántos productos hay en total.

> **Contexto Académico**: Introduce el uso de colecciones con tipos envoltorios (`Wrapper classes`) como `Double` y la acumulación de valores.

## 🎯 Objetivos de Aprendizaje
- Uso de clases Wrapper (`Double`, `Integer`).
- Acumulación de valores mediante iteración.
- Métodos de resumen de colecciones.

## 📥 Ejemplo de Entrada
```text
Precios: 15.50, 42.00, 10.25
```

## 📤 Ejemplo de Salida
```text
Total de productos: 3
Costo Total: $67.75
```

---
🕓 **Dificultad**: Fácil
📚 **Temas**: Wrapper Classes, Acumuladores, ArrayList<Double>.
