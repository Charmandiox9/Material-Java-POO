# Ejercicio 1: Registro de Estudiantes

## 📝 Descripción
Crea una clase `Estudiante` con nombre y nota. Luego, en una clase principal, gestiona una lista de objetos `Estudiante`. El programa debe permitir agregar alumnos y finalmente mostrar quién tiene la nota más alta.

> **Contexto Académico**: Salto fundamental de colecciones de tipos simples (String, Double) a colecciones de **Objetos Personalizados**. Se requiere entender cómo acceder a los métodos del objeto mientras se recorre la lista.

## 🎯 Objetivos de Aprendizaje
- Uso de `ArrayList<Objeto>`.
- Búsqueda de extremos (máximos/mínimos) dentro de una colección.
- Acceso a atributos mediante Getters durante la iteración.

## 📥 Ejemplo de Entrada
```text
Estudiantes: ("Ana", 8.5), ("Pedro", 9.2), ("Luis", 7.0)
```

## 📤 Ejemplo de Salida
```text
El mejor estudiante es Pedro con una nota de 9.2.
```

---
🕓 **Dificultad**: Intermedio
📚 **Temas**: ArrayList de Objetos, Búsqueda de máximos.
