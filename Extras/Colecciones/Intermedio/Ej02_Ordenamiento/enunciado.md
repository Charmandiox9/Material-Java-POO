# Ejercicio 2: Ordenamiento de Inventario

## 📝 Descripción
Tienes una lista de precios de productos desordenada. Utiliza las herramientas de Java para ordenar la lista de menor a mayor y mostrar el precio más barato y el más caro.

> **Contexto Académico**: Introduce la utilidad de la clase `java.util.Collections` para manipular listas de forma eficiente sin programar algoritmos de ordenamiento manualmente (como BubbleSort).

## 🎯 Objetivos de Aprendizaje
- Uso de `Collections.sort()`.
- Manipulación de listas numéricas.
- Acceso a extremos de una lista ordenada.

## 📥 Ejemplo de Entrada
```text
Precios: [50.0, 10.0, 30.0, 5.0, 20.0]
```

## 📤 Ejemplo de Salida
```text
Precios ordenados: [5.0, 10.0, 20.0, 30.0, 50.0]
Más barato: 5.0
Más caro: 50.0
```

---
🕓 **Dificultad**: Intermedio
📚 **Temas**: Collections.sort(), Ordenamiento.
