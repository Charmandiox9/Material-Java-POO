# Ejercicio 3: Depuración de Invitados

## 📝 Descripción
Se ha recolectado una lista de nombres para un evento, pero algunos nombres se repitieron por error. Debes crear un algoritmo que tome la lista original y genere una nueva lista que contenga solo nombres únicos, sin duplicados.

> **Contexto Académico**: Refuerza la lógica de comparación y transferencia de datos entre colecciones. No se permite el uso de `Set` para forzar la lógica de control manual.

## 🎯 Objetivos de Aprendizaje
- Filtrado manual de duplicados.
- Comparación de Strings (`equals`).
- Construcción de nuevas listas basadas en condiciones de existencia.

## 📥 Ejemplo de Entrada
```text
Original: ["Ana", "Pedro", "Ana", "Luis", "Pedro"]
```

## 📤 Ejemplo de Salida
```text
Lista depurada: ["Ana", "Pedro", "Luis"]
```

---
🕓 **Dificultad**: Intermedio
📚 **Temas**: Filtrado, Lógica de existencia.
