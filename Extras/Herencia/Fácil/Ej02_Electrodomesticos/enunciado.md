# Ejercicio 2: Inventario de Electrodomésticos

## 📝 Descripción
Diseña una clase `Electrodomestico` con atributos como marca y consumo energético. Luego, crea una clase `Lavadora` que herede estos atributos y añada uno específico: carga máxima.

> **Contexto Académico**: Refuerza la idea de que la herencia permite extender las capacidades de una clase existente sin duplicar código.

## 🎯 Objetivos de Aprendizaje
- Especialización de clases.
- Herencia de atributos protegidos (`protected`).
- Organización de código jerárquico.

## 📥 Ejemplo de Entrada
```text
Electrodomestico: Marca "LG", Consumo "A"
Lavadora: Carga 15kg
```

## 📤 Ejemplo de Salida
```text
Electrodoméstico: LG (Consumo: A)
Capacidad de carga: 15.0 kg
```

---
🕓 **Dificultad**: Fácil
📚 **Temas**: Especialización, Atributos heredados.
