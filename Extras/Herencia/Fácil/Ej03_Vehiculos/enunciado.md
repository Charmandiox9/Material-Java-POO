# Ejercicio 3: Tipos de Vehículos

## 📝 Descripción
Crea una clase `Vehiculo` con el atributo `velocidadMaxima`. Desarrolla dos subclases: `Coche` (con número de puertas) y `Bicicleta` (con tipo de terreno: montaña o ruta). 

> **Contexto Académico**: Se enfoca en cómo diferentes ramas de una jerarquía pueden tomar caminos distintos basándose en una base común.

## 🎯 Objetivos de Aprendizaje
- Ramificación de jerarquías.
- Atributos específicos por subclase.
- Uso consistente de constructores.

## 📥 Ejemplo de Entrada
```text
Coche: Vel 180 km/h, 5 puertas
Bicicleta: Vel 40 km/h, Terreno: "Montaña"
```

## 📤 Ejemplo de Salida
```text
Vehículo a 180.0 km/h con 5 puertas.
Bicicleta de Montaña a 40.0 km/h.
```

---
🕓 **Dificultad**: Fácil
📚 **Temas**: Jerarquías múltiples, Atributos específicos.
