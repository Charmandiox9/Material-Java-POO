# Ejercicio 2: Polifonía Animal

## 📝 Descripción
Diseña una clase `Mascota` con un método `hacerSonido()`. Crea subclases `Perro`, `Gato` y `Pajaro`. Cada una debe emitir su propio sonido. En el main, utiliza un arreglo de tipo `Mascota` para demostrar cómo el mismo método se comporta de forma distinta (Polimorfismo).

> **Contexto Académico**: Se enfoca en el polimorfismo dinámico: tratar a diferentes objetos como si fueran del mismo tipo padre pero ejecutando el comportamiento específico de cada uno.

## 🎯 Objetivos de Aprendizaje
- Polimorfismo de métodos.
- Uso de arreglos de superclases.
- Concepto de "enlace dinámico".

## 📥 Ejemplo de Entrada
```text
Lista de mascotas: [Perro, Gato, Pajaro]
```

## 📤 Ejemplo de Salida
```text
El perro dice: ¡Guau!
El gato dice: ¡Miau!
El pájaro dice: ¡Pío!
```

🕓 **Dificultad**: Intermedio
