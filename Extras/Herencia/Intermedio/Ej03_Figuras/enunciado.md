# Ejercicio 3: Calculadora de Figuras Herencia

## 📝 Descripción
Crea una clase `Figura` con un color. Deriva clases `Circulo` (con radio) y `Rectangulo` (con base y altura). Ambas deben sobrescribir un método `calcularArea()`. 

> **Contexto Académico**: Practica la herencia aplicada a modelos matemáticos, donde la superclase provee la estructura común (color) y las subclases la lógica de cálculo.

## 🎯 Objetivos de Aprendizaje
- Uso de fórmulas matemáticas en métodos sobrescritos.
- Constante `Math.PI`.
- Reutilización de constructores complejos.

## 📥 Ejemplo de Entrada
```text
Círculo: Rojo, Radio 5.0
Rectángulo: Azul, Base 4, Altura 3
```

## 📤 Ejemplo de Salida
```text
Círculo Rojo - Área: 78.54
Rectángulo Azul - Área: 12.0
```

🕓 **Dificultad**: Intermedio
📚 **Temas**: Sobrescritura, Math, Geometría POO.
