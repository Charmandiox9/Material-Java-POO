# Ejercicio 3: Lógica de Tablero 3x3 (Tres en Línea)

## 📝 Descripción
Debes desarrollar la lógica base para un juego de **Tres en Línea (Gato)** utilizando una matriz de 3x3. El programa debe permitir que dos jugadores (Jugador 1 y Jugador 2) ingresen sus coordenadas (fila y columna) por turnos para marcar sus movimientos. El programa debe validar que la posición esté disponible y, tras cada jugada, verificar si el jugador actual ha completado una fila o una columna.

> **Contexto Académico**: Este ejercicio es un reto de lógica que combina el manejo de matrices, validación de rangos y la implementación de algoritmos de detección de patrones en estructuras bidimensionales.

## 🎯 Objetivos de Aprendizaje
- Implementación de juegos por turnos mediante lógica de ciclos.
- Validación de coordenadas dentro de los límites de una matriz (0-2).
- Desarrollo de algoritmos de verificación (recorrido de filas y columnas).
- Manejo de estados en una matriz (0: Vacío, 1: Jugador X, 2: Jugador O).

## 📥 Ejemplo de Entrada
```text
Turno Jugador 1:
- Fila: 0
- Columna: 0

Turno Jugador 2:
- Fila: 1
- Columna: 1
```

## 📤 Ejemplo de Salida
```text
[1][0][0]
[0][2][0]
[0][0][0]

¡Movimiento válido! Siguiente turno.
```

---
🕓 **Dificultad**: Difícil
📚 **Temas**: Matrices 3x3, Lógica de Juego, Validación de Estados.
