# Ejercicio 1: Sistema de Evaluación Académica

## 📝 Descripción
Eres un profesor que necesita automatizar el cálculo de promedios de sus alumnos. El programa debe solicitar la cantidad de notas de un estudiante, permitir el ingreso de cada nota y, al finalizar, mostrar el promedio final y determinar si el alumno está **Aprobado** (nota >= 4.0) o **Reprobado** (nota < 4.0).

> **Contexto Académico**: Este ejercicio refuerza el uso de acumuladores, contadores y la lógica de decisión simple.

## 🎯 Objetivos de Aprendizaje
- Uso de la clase `Scanner` para entrada de datos.
- Implementación de ciclos `for` para recolección de datos.
- Aplicación de condicionales `if-else` para lógica de negocio.
- Manejo de tipos de datos `double` y casting básico.

## 📥 Ejemplo de Entrada
```text
Cantidad de notas: 3
Nota 1: 5.5
Nota 2: 4.0
Nota 3: 6.2
```

## 📤 Ejemplo de Salida
```text
=== RESULTADO ACADÉMICO ===
Promedio Final: 5.23
Estado: APROBADO
```

---
🕓 **Dificultad**: Fácil
📚 **Temas**: Variables, Ciclo for, Condicionales.
