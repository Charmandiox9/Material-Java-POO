# Ejercicio 2: Conversor de Divisas

## 📝 Descripción
Un banco necesita una aplicación sencilla para que sus clientes conviertan montos en pesos chilenos (CLP) a dólares (USD) o euros (EUR). Debes desarrollar un programa que muestre un menú interactivo y permita realizar múltiples conversiones hasta que el usuario decida salir.

> **Tipos de Cambio ficticios**:
> - 1 USD = 850 CLP
> - 1 EUR = 920 CLP

## 🎯 Objetivos de Aprendizaje
- Implementación de menús con ciclos `do-while`.
- Uso de la estructura `switch-case` para selección de opciones.
- Manejo de constantes y cálculos matemáticos simples.
- Validación de salida de programas.

## 📥 Ejemplo de Entrada
```text
=== CONVERSOR DE DIVISAS ===
1. CLP a USD
2. CLP a EUR
0. Salir
Opción: 1
Monto en CLP: 10000
```

## 📤 Ejemplo de Salida
```text
Monto convertido: 11.76 USD
```

---
🕓 **Dificultad**: Fácil
📚 **Temas**: Ciclo do-while, Switch-case, Variables.
