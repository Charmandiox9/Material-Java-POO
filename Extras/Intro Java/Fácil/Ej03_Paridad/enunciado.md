# Ejercicio 3: Análisis de Números

## 📝 Descripción
Un investigador estadístico necesita una herramienta para clasificar una serie de números enteros. Debes desarrollar un programa que solicite números al usuario de forma continua hasta que se ingrese un número negativo. Al finalizar, el programa debe mostrar cuántos números fueron **pares**, cuántos **impares** y cuántos fueron exactamente **cero** (sin contar el número negativo de salida).

> **Contexto Académico**: Este ejercicio refuerza el uso del operador residuo (%) y el manejo de acumuladores/contadores en ciclos de fin no determinado.

## 🎯 Objetivos de Aprendizaje
- Uso de ciclos `while` con condición de salida dinámica.
- Aplicación del operador módulo (`%`) para determinar paridad.
- Manejo de múltiples contadores de estado.
- Lógica de control de flujo anidada (if-else dentro de while).

## 📥 Ejemplo de Entrada
```text
Ingrese un número (negativo para salir): 10
Ingrese un número: 0
Ingrese un número: 7
Ingrese un número: -1
```

## 📤 Ejemplo de Salida
```text
=== RESUMEN ESTADÍSTICO ===
Cantidad de Pares: 1
Cantidad de Impares: 1
Cantidad de Ceros: 1
```

---
🕓 **Dificultad**: Fácil
📚 **Temas**: Ciclo while, Operador módulo, Contadores.
