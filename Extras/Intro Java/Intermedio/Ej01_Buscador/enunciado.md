# Ejercicio 1: Buscador de Elemento en Vector

## 📝 Descripción
Un almacén de repuestos necesita una herramienta para verificar si un código de producto se encuentra en su inventario actual. Debes desarrollar un programa que solicite al usuario la cantidad de productos a registrar, almacene sus códigos (números enteros) en un **vector** y luego permita realizar una búsqueda. Si el código existe, debe mostrar su **posición (índice)**; de lo contrario, informar que no se encontró.

> **Contexto Académico**: Este ejercicio introduce el manejo de arreglos unidimensionales y la técnica de búsqueda lineal.

## 🎯 Objetivos de Aprendizaje
- Declaración e inicialización de arreglos (vectores).
- Uso de ciclos `for` para el llenado y recorrido de arreglos.
- Implementación de banderas (flags) para control de búsqueda.
- Manejo de índices y acceso a memoria estática.

## 📥 Ejemplo de Entrada
```text
Cantidad de productos: 4
Código 1: 105
Código 2: 202
Código 3: 308
Código 4: 415

Código a buscar: 308
```

## 📤 Ejemplo de Salida
```text
Código encontrado en la posición: 2
```

🕓 **Dificultad**: Intermedio
📚 **Temas**: Vectores, Búsqueda Lineal, Ciclos.
