# Ejercicio 2: Ordenamiento de Precios (BubbleSort)

## 📝 Descripción
Un administrador de una tienda de electrónica necesita organizar los precios de sus productos de menor a mayor para generar un reporte de ventas. Debes desarrollar un programa que permita ingresar una lista de precios (números decimales), almacene estos valores en un vector y los ordene utilizando el algoritmo **BubbleSort**. Además, el programa debe mostrar el precio más bajo, el más alto y la lista completa ya ordenada.

> **Contexto Académico**: Este ejercicio refuerza el uso de ciclos anidados para la implementación de algoritmos de ordenamiento y el intercambio (swap) de valores en memoria.

## 🎯 Objetivos de Aprendizaje
- Implementación académica del algoritmo de **Ordenamiento Burbuja**.
- Uso de variables auxiliares para el intercambio de valores.
- Cálculo de estadísticas básicas (mínimo y máximo) sobre un vector.
- Visualización de datos procesados.

## 📥 Ejemplo de Entrada
```text
Cantidad de precios: 5
Precio 1: 1500.5
Precio 2: 990.0
Precio 3: 2100.2
Precio 4: 550.0
Precio 5: 1200.0
```

## 📤 Ejemplo de Salida
```text
=== REPORTE DE PRECIOS ===
Precio Mínimo: 550.0
Precio Máximo: 2100.2

Precios Ordenados:
550.0 - 990.0 - 1200.0 - 1500.5 - 2100.2
```

🕓 **Dificultad**: Intermedio
📚 **Temas**: Vectores, BubbleSort, Ciclos anidados.
