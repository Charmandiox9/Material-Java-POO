# Ejercicio 3: Descomposición de Dígitos

## 📝 Descripción
Un analista de datos necesita procesar números de identificación largos para realizar validaciones de seguridad. Debes desarrollar un programa que solicite un número entero positivo al usuario, descomponga el número en sus **dígitos individuales** y los almacene en un vector. Finalmente, el programa debe mostrar cada dígito por separado y calcular la **suma total** de los mismos.

> **Contexto Académico**: Este ejercicio refuerza el uso de los operadores residuo (%) y división entera (/) para la extracción de componentes numéricos, además de la gestión dinámica de arreglos basada en el tamaño del dato de entrada.

## 🎯 Objetivos de Aprendizaje
- Uso de operadores aritméticos para descomposición numérica.
- Implementación de algoritmos para determinar la longitud de un número.
- Almacenamiento y recuperación de datos en vectores.
- Acumulación de valores mediante iteración.

## 📥 Ejemplo de Entrada
```text
Ingrese un número entero positivo: 4508
```

## 📤 Ejemplo de Salida
```text
Dígitos detectados: 4, 5, 0, 8
Suma total: 17
```

---
🕓 **Dificultad**: Intermedio
📚 **Temas**: Operadores matemáticos, Ciclo while, Vectores.
