# Material Java y POO

![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=openjdk&logoColor=white)

Este repositorio contiene el material de las ayudantías que fueron realizadas para el curso de **Programación Orientada a Objetos (POO)**, incluyendo ejercicios prácticos, soluciones, diagramas y recursos de apoyo.

Actualmente, el enfoque principal está en la modelación y resolución de nuevo ejercicios externos al curso antes mencionado, por lo tanto lo que se verá **puede** que no se vea en el curso, y su uso **puede** estar **prohibido**.

## 📄 Estructura de cada ejercicio

Cada ejercicio se encuentra en su propia carpeta e incluye:

- `enunciado.md`: Descripción del problema y objetivos.
- `solucion.java`: Código de solución en Java.
- `datos.txt`: (opcional) Datos de prueba o entrada.
- `diagrama.png` / `diagrama.drawio`: (si aplica) Diagrama de clases o flujo.
